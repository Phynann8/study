3.Break Points

Break points គឺប្រើសម្រាប់ផ្អាក់ដំណើរការ Code នៅLine Statement ណាមួយដើម្បីពិនិត្យCode នៅពេលមួយនោះ.

Action Keystroke Description
Break Point F9 បញ្ឈប់ដំណើរការនៃកូដនៅត្រង់ Line Statement ណាមួយ
Continue Code Execution F5 បន្តដំណើការបន្ទាប់ពី Break Point
Step Into F11 សំរាប់បង្ហាញ Line Statement ដែលដំណើការបន្ទាប់ពី Break Point ហើយចូលទៅកាន់ Sub នៃ Line Statement នីមួយៗ
Step Over F10 សំរាប់បង្ហាញ Line Statement ដែលដំណើការបន្ទាប់ពី Break Point ប៉ុន្តែមិនចូលទៅកាន់ Sub នៃ Line Statement នីមួយៗនោះទេ
Step Out Shift+F11 សំរាប់បង្ហាញ Line Statement ទាំងអស់
Stop Debugging Shift+F15 បញ្ចប់ដំណើការ Break Point និងដំណើការ ​Code
