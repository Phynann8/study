4.Try Catch Finally

Try Catch Finally គឺប្រើសម្រាប់ពិនិត្យកំហុសដែលអាចកើតមានឡើងនៅក្នុងCode

Try Block

Try ប្រើសំរាប់កំណត់ប្រភេទកូដដែលអាចបណ្តាលឲ្យកើតមានករណី exception (Runtime-Error)។

ករណី​មាន Error កូដនិងបន្តទៅដំណើការ Catch Block

តែបើសិនមិនមាន Error ទេ កូដនៅក្នុង Try Block នឹងដំណើការធម្ដារ
