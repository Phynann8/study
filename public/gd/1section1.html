1.Adding Comments

យើងអាចប្រើប្រាស់ Comments ដើម្បីធ្វើការកត់ចំណាំ ឬសំគាល់ជាអត្ថបទខ្លីៗនៅក្នុង Program Code ដោយវាមិនត្រូវ បាន Execute ឡើយ។ នៅក្នុង ភាសា C# មាន Comments ពីរប្រភេទសំរាប់ឲប្រើប្រាស់ដែលមានដូចជា:

Simgle Comment:យើងប្រើប្រាស់ two forward slashes (//) នៅខាងមុខអក្សរទាំងឡាយដែលគិតថាជា Comments.

Ex:
  // This is a comment because it is preceded by double forward slashes.
  int intAge; // Used to store the user's age in years.

Multiple Comment:ប្រើប្រាស់ asterisk (*) នៅខាងមុខ អក្សរ Comments ហើយបញ្ចប់ជាមួយនឹង forward slash (*/).

Ex:
  /*This is how to can comment multiple lines of code.
  This is the second line of the comment.*/
