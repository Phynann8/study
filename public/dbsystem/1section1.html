I.Database Files

SQL Server databasees have three types of files:

1- Primary data Files

The Primary data files is the starting pint of database and points to the other files in the database.

Every Database has one primary data file.

The recommended file name extension for primary data file is .mdf

2.Secondary data file

Secondary Data Files make up all the data file, other than the primary data file.

Some databse may not have any Secondary data files,while other have several secondary date files

The recommended file name extension for socondary data file is .ndf
